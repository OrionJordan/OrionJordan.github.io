Limits of Functions

Definition

Let f \colon D \to \mathbb{R} and let c be an accumulation point of D. We say that a real number L is a limit of f at c if

\forall \varepsilon > 0, \exists \delta > 0\; s.t.\; 0 < |x-c| < \delta \implies |f(x) - L| < \varepsilon

Theorem

Let f \colon D \to \mathbb{R} and c be an accumulation point of D. Then \lim_{x\to c}f(x) = L if and only if for each neighborhood of V of L there exists a deleted neighborhood U of c such that f(U \cap D) \subseteq V.

Theorem

Let f \colon D \to \mathbb{R}, and c be an accumulation point of D. Then, \lim_{x\to c} f(x) = L if and only if for every sequence (s_n) in D that converges to c with s_n \not = c for all n, the sequence (f(s_n)) converges to L.

Proof

Let (s_n) be a sequence in D that converges to c and s_n \not = c \forall n, and \lim_{x\to c} f(x) = L.

We want to show that \lim_{n\to \infty} f(s_n) = L.

By assumption \lim_{x\to c} f(x) = L, so given any \varepsilon > 0, \exists \delta > 0\;s.t.\; 0 < |x-c| < f \implies |f(x) - L| < \varepsilon by assumption.

Since S_n \to c, \exists N \in \mathbb{N}\;s.t.\; \forall n \geq N, we have that |s_n - c| < \delta. Now we have that \forall n \geq N, |f(s_n) - L| < \varepsilon thus (f(s_n)) \to L.

Theorem

Let f \colon D \to \mathbb{R}, and c be an accumulation point of D. Then the following are equivalent

  • f does not have a limit at c

  • There exists a sequence (s_n) in D with each s_n \not = c such that (s_n) converges to c, but (f(s_n)) is not convergent in \mathbb{R}.

Example

Let f(x) = \sin(1/x) for x > 0. We want to show that \lim_{x\to 0} \sin(1/x) does not exists.

We prove this by considering two sequences, s_n = \frac{2}{\pi n}, then s_n \to 0 as \frac{1}{n} \to 0.

Definition

Let f \colon D \to \mathbb{R}, where c \in D. We say that f is continous at c if

\forall \varepsilon > 0, \exists \delta > 0, s.t.\; x \in D and |x-c| < \delta then $|f(x) - f(c)| < \varepsilon$.

If f is continous at all points x \in D, we say that f is continouos on D.
